# 数据字段说明

## 一、Mock生成的原始数据

### 1.1 商品基础信息 (products.json)

模拟从抖音平台采集的商品公开数据。

| 字段 | 类型 | 示例值 | 说明 |
|------|------|--------|------|
| product_id | string | P000001 | 商品唯一标识 |
| name | string | OPPO Find X7 | 商品名称（品牌+型号+后缀） |
| category | string | 手机 | 品类，共8类：手机/电脑/服装/美妆/食品/家居/数码/运动 |
| brand | string | OPPO | 品牌名称 |
| price | float | 3693.18 | 当前售价（元） |
| original_price | float | 4599.00 | 原价（比price高10%-50%） |
| image_url | string | https://example.com/... | 商品主图URL（占位地址） |
| description | string | OPPO旗舰手机，拍照效果出色... | 商品卖点描述（LLM模板生成） |
| shop_name | string | OPPO手机旗舰店 | 店铺名称 |
| create_time | string | 2025-12-12T20:53:16 | 商品上架时间 |

**生成脚本：** `mock-data/generate_products.py`
**数据量：** 100条（8个品类各12-13个）

---

### 1.2 日销量数据 (monthly_sales.json)

模拟每天从抖音商品页采集的销量数据，用于Spark聚合计算趋势。

| 字段 | 类型 | 示例值 | 说明 |
|------|------|--------|------|
| product_id | string | P000001 | 关联商品ID |
| date | string | 2025-06-17 | 日期（YYYY-MM-DD） |
| daily_sales | int | 156 | 当日销量（件） |
| daily_amount | float | 779844.00 | 当日销售额（元）= daily_sales × price |

**生成脚本：** `mock-data/generate_monthly_sales.py`
**数据量：** 36,500条（100商品 × 365天）
**生成策略：**
- 基础日销量：10-500件/天
- 星期效应：周末比工作日高20-50%
- 月度趋势：部分商品上升(5-15%/月)，部分下降(3-10%/月)
- 大促活动：双11(4-6倍)、618(3-5倍)、年货节(2.5-4倍)

---

### 1.3 评价数据 (reviews.json)

模拟用户在抖音商品页的评价，由NLP引擎从模板库生成。

| 字段 | 类型 | 示例值 | 说明 |
|------|------|--------|------|
| review_id | string | R000001_001 | 评价唯一标识 |
| product_id | string | P000001 | 关联商品ID |
| content | string | 质量很好，面料舒适，物流也快 | 评价内容（NLP模板+同义词替换生成） |
| rating | int | 5 | 评分（1-5星） |
| sentiment | string | positive | 情感标签：positive/negative/neutral |
| likes | int | 12 | 其他用户点赞数 |
| username | string | 小明 | 评价用户名 |
| create_time | string | 2025-08-15T14:30:00 | 评价时间 |

**生成脚本：** `mock-data/generate_reviews.py`（调用 `nlp_generator.py`）
**数据量：** 约1,200条（每商品8-15条）
**评分分布：** 好评(4-5星) 60% + 中评(3星) 20% + 差评(1-2星) 20%
**模板来源：** `mock-data/templates/` 下4个JSON文件（LLM生成）

## 二、Spark计算的分析指标

### 2.1 商品分析结果 (product_analysis)

由 `spark-jobs/product_analysis_job.py` 或 `scripts/compute_analysis.py` 计算。

| 字段 | 类型 | 计算公式 | 说明 |
|------|------|---------|------|
| hot_score | float | (0.4×sales_score + 0.3×growth_score + 0.2×rating_score + 0.1×review_score) × 1000 | 爆款指数，0-1000分 |
| total_sales | int | SUM(daily_sales) | 近12个月总销量 |
| avg_rating | float | AVG(rating) | 平均评分 |
| review_count | int | COUNT(reviews) | 评价总数 |
| positive_rate | float | COUNT(rating>=4) / total | 好评率（0-1） |
| negative_rate | float | COUNT(rating<=2) / total | 差评率（0-1） |
| monthly_growth | float | (当月销量-上月销量)/上月销量 × 100 | 月增长率（%） |
| top_tags | list | jieba提取评价高频词Top10 | 关键词标签 |
| sales_trend | list | 近12个月每月总销量数组 | 销量趋势 |

**爆款指数细分公式：**
- sales_score = 当月销量 / 品类最高销量（归一化0-1）
- growth_score = min(月增长率/100%, 1.0)
- rating_score = avg_rating / 5.0
- review_score = min(review_count / 500, 1.0)

### 2.2 评价分析结果 (review_analysis)

由 `spark-jobs/nlp_analysis_job.py` 计算。

| 字段 | 类型 | 说明 |
|------|------|------|
| high_freq_keywords | string(JSON) | 高频关键词列表 |
| sentiment_distribution | string(JSON) | 情感分布 {"positive": N, "negative": N, "neutral": N} |
| rating_distribution | string(JSON) | 评分分布 {"1": N, "2": N, ..., "5": N} |

### 2.3 趋势数据 (trend_data)

由 `spark-jobs/trend_aggregation_job.py` 计算。

| 字段 | 类型 | 说明 |
|------|------|------|
| monthly_sales | string(JSON) | 按月聚合的销量数组 |
| category_trend | string(JSON) | 按品类的月度趋势 |

## 三、数据流转图

```
Mock生成                    Spark计算                    前端展示
─────────                  ─────────                   ─────────

products.json ──────────┐
  (100条商品)           │
                        ├──→ product_analysis ────→ 爆款分析Tab
monthly_sales.json ─────┤     (hot_score等)
  (36500条日销量)       │
                        │
reviews.json ───────────┘
  (约1200条评价)            ├──→ review_analysis ───→ 差评分析Tab
                               (关键词/情感)
```

## 四、HBase表结构

| 表名 | RowKey | Column Family | 说明 |
|------|--------|--------------|------|
| product | product_id | info | 商品基础信息（mock生成） |
| monthly_sales | product_id_date | data | 日销量原始数据（mock生成） |
| review | review_id | info | 评价原始数据（mock生成） |
| product_analysis | product_id | metrics, trend | Spark计算的商品分析结果 |
| review_analysis | product_id | stats | Spark计算的评价分析结果 |
| copywriting | copywriting_id | info | AI生成的文案 |
